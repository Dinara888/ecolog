# 🌍 EcoLog — Экологический дневник

Консольное приложение для учёта экопривычек, аналитики и мотивации.

## 📦 Установка
1. `python -m venv venv`
2. `source venv/bin/activate` (или `venv\Scripts\activate` на Windows)
3. `pip install -r requirements.txt`

## 🚀 Запуск
`python src/main.py`

## 📁 Структура
- `src/` — исходный код (модули: config, db, analytics, data_manager, main)
- `backups/` — автосохранённые копии БД
- `.env` — конфигурация путей и логирования

## 🛠 Технологии
Python 3.8+, SQLite, JSON, CSV, ZIP, pathlib, logging, python-dotenv
